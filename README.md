# Casa Assistiva
Projeto para a disciplina SPOTTI1.

# Escopo
O projeto estrutura um circuito que tera funcionalidades relacionadas a seguranca, como deteccao de queda, deteccao de gases prejudiciais a saude, alarme de invasao, etc.
